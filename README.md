## 📌 Описание проекта
StudyBuddy — это интерактивная посадочная страница для приложения, помогающего студентам планировать учебные задачи, управлять дедлайнами и отслеживать прогресс.

Проект разработан в рамках учебной дисциплины «Тенденции проектирования графических пользовательских интерфейсов» и демонстрирует навыки UX/UI проектирования и frontend-разработки.

---

## 🎯 Цель проекта
Создать современный адаптивный лендинг, который:
- быстро объясняет ценность продукта
- удерживает внимание пользователя через интерактив
- мотивирует оставить заявку / скачать приложение
- обеспечивает высокий уровень конверсии

---

## 👥 Целевая аудитория

### Основная аудитория:
- Студенты 18–30 лет
- Бакалавры, магистранты
- Активные пользователи мобильных устройств
- Испытывают трудности с планированием задач

### Вторичная аудитория:
- Молодые специалисты
- Люди, совмещающие работу и обучение

---

## 💡 Продуктовая гипотеза
Если создать простой и визуально понятный лендинг с акцентом на CTA, интерактивными блоками и минимальной формой регистрации, то пользователь быстрее поймёт ценность продукта и с большей вероятностью оставит заявку.

---

## 🧠 UX/UI решения
- Mobile-first подход (основной трафик — мобильные устройства)
- CTA размещён в Hero-блоке
- Минимальная форма (только имя и email)
- Интерактивные элементы (аккордеон, модальные окна)
- Социальное доказательство (цифры и отзывы)
- Плавные анимации для повышения вовлечённости

---

## ⚙️ Функционал
- Бургер-меню для мобильной версии
- Плавная прокрутка по якорям
- Аккордеон в блоке "Функции"
- Модальное окно
- Валидация формы регистрации
- Анимации появления элементов (scroll reveal)
- Hover-анимации карточек и кнопок

---

## 📊 Социальное доказательство
- 5000+ студентов используют приложение
- Рейтинг 4.8 / 5
- 95% пользователей довольны результатом

---

## 🛠️ Технологии
- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, адаптив)
- JavaScript (DOM, события, валидация)
- Без сторонних UI-фреймворков
